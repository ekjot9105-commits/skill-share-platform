# Skill-Sharing Platform 🚀

A modern, localized, peer-to-peer marketplace connecting skilled workers with community members in need of services.

---

## 📜 Overview
This platform provides a unified environment for discovering local talent, managing service bookings, and facilitating real-time communication. From independent plumbers to freelance designers, we empower workers to grow and clients to find reliable help.

> [!TIP]
> For a detailed look at the project's use cases, technology stack, and architecture, please refer to the **[Platform Overview](PLATFORM_OVERVIEW.md)**.

---

## 🛠️ Tech Stack
- **Frontend**: React 19, TypeScript, Vite, Tailwind CSS, Framer Motion, Zustand.
- **Backend**: Node.js, Express 5, SQLite3, Socket.io.

---

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher)
- [npm](https://www.npmjs.com/) (v9 or higher)

### Installation
1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd skill-sharing-platform-project
   ```

2. **Setup Frontend**:
   ```bash
   npm install
   ```

3. **Setup Backend**:
   ```bash
   cd backend
   npm install
   ```

---

## 🏃 Running the Application

### 1. Start the Backend Server
From the `backend` directory:
```bash
npm start
```
*Port 3000 will be used.*

### 2. Start the Frontend
From the root directory:
```bash
npm run dev
```
*Typically runs on [http://localhost:5173/](http://localhost:5173/).*
